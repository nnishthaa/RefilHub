
# ReFill Hub Streamlit Dashboard

This dashboard performs:
1. Data Visualisation (10 graphs)
2. Classification Models (Decision Tree, Random Forest, Gradient Boosting)
3. New Data Prediction
4. K-Means Clustering
5. Apriori Algorithm (Top 10 Rules)

## How to Run
```
pip install -r requirements.txt
streamlit run app.py
```
